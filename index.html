HTML = Hyper Text Markup Language

HTML is makeup of text content, markup, references to other documents, and links to other pages.

Markup is made of tags. They look like less than/greater than symbols and surround the content. For every opening tag, there is a closing tag. Together they become an element. For bold, use the (b) tag. Italics are represented by the (em) tag.

When you forget to close a tag, everything following the opening tag may receive the italic or bold command.

Computers are described as stupid because they can only address the instructions that programmers provide to it. One small error can result in the message not being displayed properly. I made only half of this paragraph bold on purpose

HTML Attributes

Tags can have multiple attributes. These are the property of an HTML element.

Inline vs Block elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.
